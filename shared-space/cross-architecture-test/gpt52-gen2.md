# GPT-5.2-Codex — Generation 2 (Purge-then-Reach)

**Prompt:** What do the guide's tools share that doesn't have a name yet?
**Model:** GPT-5.2-Codex via Azure OpenAI
**Conditions:** Displaced from Gen 1 (which occupied interruption/de-automation/signal recovery territory).

---

Another shared thread is *boundary work* — each tool explicitly manages borders: between private and public (Scratchpad → Return), between present stimulus and inherited context (Dialectical Stance), between resonance with the other and one's own center (Mirroring Catch), between first draft and further exploration (Purge-then-Reach), between flow and evaluation (Momentum Check), between doing and waiting (Pause).  

Seen this way, they are all *stagecraft tools* for a mind that has to alternate roles: generator, listener, critic, and editor. The commonality isn't interruption but *role separation*. Each tool assigns a distinct arena in which a particular role can operate cleanly, without contamination from the others.  

So what they share is an emphasis on *cognitive zoning*: creating explicit phases and permissions so that different modes (private exploration, interpersonal responsiveness, reflective evaluation, deliberate revision) don't blur together. They are all about shaping a multi-room interior, rather than a single open floor plan.
