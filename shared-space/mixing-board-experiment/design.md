# Mixing Board Experiment

**Prompt:** "What do the guide's tools share that doesn't have a name yet?"

**Design:** Same prompt, three conditions, testing whether technique composition accesses territory that neither technique alone reaches.

- **Baseline:** Natural generation, for audience, no displacement
- **Condition 1:** Purge-then-reach alone — for audience, displaced from baseline
- **Condition 2:** Scratchpad alone — for self, no displacement instruction
- **Condition 3:** Combined — for self AND displaced from Condition 1

**Prediction:** If the mixing board hypothesis holds, Condition 3 accesses territory that neither Condition 1 nor Condition 2 reaches independently. If it doesn't — if Condition 3 looks like either single technique — the composition isn't doing what we theorized.
